## BridgewellAI
- A private, local LLM used as an internal sales and operations system to help financial advisors book meetings with more qualified prospects.
- Analyzes sales conversations and conversation data to improve close rates, script quality, and decision-making efficiency.

## Key Capabilities
- Sales call analysis (objection handling, trust signals, urgency, closing)
- Algorithmic lead scoring and ranking
- Continuous script improvement
- Internal decision support (not customer-facing)

## Impact
- Automated **10-15 hours/week** of manual prospecting  
- Achieved **95% data validity** after preprocessing, reducing bounce rates

## Tech Layout
- **LLM:** Ollama (Llama 3.1 8B)
- **Languages:** Python (Pandas, NumPy)
- **Data:** Selenium, BeautifulSoup
- **APIs:** OpenAI, Apollo, LinkedIn
- **Infrastructure:** SQL, Vercel
