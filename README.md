# Technical Architecture
The system follows a 5-stage Data Science lifecycle:
1. Data Acquisition:
Scrapes public financial data using BeautifulSoup and Selenium. 
2. Integration:
We use Apollo.io and LinkedIn APIs for contact verification.
3. Preprocessing & Wrangling:
Cleans unstructured data using Pandas & Implements logic to remove duplicates and validate email syntax/domain health.
4. Algorithmic Filtering:
Applies a weighted scoring algorithm to rank leads based on client-defined AUM and location criteria.
5. Automated Outreach:
Triggers AI-personalized email sequences with SMTP and OpenAI API for context-aware messaging.
